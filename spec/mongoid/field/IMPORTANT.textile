h1. Mongoid geo field

For @Mongoid::Field.option :geo@

We should leverage the parsing capabilities og geo_calc and GeoPoint. A GeoPoint can have a #coord_mode set to :lng_lat. 
Also, it extends various core classes with a #geo_point method which converts to a 2-element array and takes into account the global #coord_mode setting for this!

Example usage:

<pre>
  GeoPoint.coord_mode = :lng_lat

  "23, 45".geo_point
    => [23, 45]

  location.lng 
    => 23


  [23, 45].geo_point 
    => [23, 45]

  location.lng 
    => 23  
</pre>
